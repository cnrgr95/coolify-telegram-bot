# Telegram Botu

## Mevcut Menüler

- Uygulamalar
- Sistem Durumu
- Veritabanları
- Zamanlanmış İşler
- Telegram Yetkileri
- Web Kullanıcıları
- Web Panel bağlantısı

## Yönetim

Uygulama ekranında durum, log, deploy, redeploy, restart, stop, silme ve zamanlama işlemleri rol izinlerine göre gösterilir. Serbest giriş akışlarında `Geri`, `İptal` ve `/iptal` kullanılabilir.

Tehlikeli işlemlerin tamamına tutarlı iki aşamalı onay eklenmesi Faz 0 güvenlik işidir. Hedef komut parity listesi `MIGRATION.md` içinde takip edilir.
