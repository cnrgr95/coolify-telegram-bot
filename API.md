# API

Mevcut sürüm genel amaçlı, sürümlenmiş bir backend API sunmamaktadır. `/api/dashboard` yalnız oturum açmış web paneli için envanter özeti döndürür.

Hedef API aşağıdaki kaynakları kapsayacaktır:

- `/api/v1/servers`
- `/api/v1/applications`
- `/api/v1/services`
- `/api/v1/containers`
- `/api/v1/deployments`
- `/api/v1/metrics`
- `/api/v1/alerts`
- `/api/v1/audit-events`

Endpoint contract'ları ortak servis katmanı kurulmadan sahte olarak yayınlanmayacaktır.
